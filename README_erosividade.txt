README_erosividadechuva

Esta é uma tentativa de atualizar e refazer de forma mais fina a layer de erosividade.
A ideia é usar o paper do Almeida et al 2019 e de Melo et al 2015.

Já temos um script feito usando as regressões de RUSLE (Melo et al 2015), 
porém gostaria de fazer essa layer de outra forma pra testar se vai ficar melhor.

Pra isso vou combinar as indicações de ambas as referências.

O objetivo para a disciplina é deixar o script e as pastas de forma reprodutível.

O output é um raster dataset, portanto não deve ser adicionado ao github. Apenas a tabela de equações e 
os scripts devem ser monitorados.
